TO RUN REINFORCEMENT LEARNING AGENT WITH ALE:

cd into rl-qbert folder

run commmand : python qbert-agent-ale.py <SEED> <USE_PRIOR> <SHOW_SCREEN>
<SEED> : Enter custom int seed (i.e. 456 or 123 or 236 or 363)
<USE_PRIOR> :type 'use_prior'to enable optimistic prior and disable random e-greedy approach, else type 'disable'
<SHOW_SCREEN> : type 'set_screen' to visualise QBert in the game, ele type 'disable
